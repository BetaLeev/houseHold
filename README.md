#### 
品牌文化

| 溯 古 |

倒叙千载后，溯古独知心

雅望|古人审美

中国传统文化体系里，审美之源溯由已久。孟子以『充实之为美』，荀子以『不全不粹之不足以为美』，老子曰『大音希声，大象无形』，庄子言『朴素而天下莫能与之争美』，苏轼语『天工与清新』，清静、纯粹、无形、朴素……诸种美好属性构造了古人“雅”生活的主线脉。

雅望古贤，晋人之风骨、宋人之风雅、明人之风流，皆为后世所追慕，古人性美，澹然若春山之云，安而能静；洁然如秋江之水，流之愈清。怀古提炼的雅生活，是风雅，是正雅，其中关于居筑、造物与安身、处世的生活哲学，更是中国古人传承下来的审美智慧。

遥集|古人志趣

中国历史上的文人士族，面对承载文字的摩崖碑版、甲骨陶壁，穷极宇宙阴阳黑白的物理真相，以纸墨传续的供养仪式，表达对精神的景仰和对文明的传承。梅兰菊竹，因为符合儒家的“君子”追求，成为历代文人墨客钟爱的对象；山水泉林，因为是清虚恬淡的理想之境，遂成为出世之士心爱的栖身之所。诗文书画以精神为主，诗者言志，文以心学，书画尚意，人化自然，只有『意深』才能『味有余』。采集古人『志趣』说，『意』『趣』二字凝结了中国古代最本真的精神追求。

| 师 古 |

闭门事古学，妙处倘可窥

造诣|古人艺术

传统家具作为传承中国艺术的重要载体，在几千年的发展历程中，深深的烙上了宗教哲学思想、建筑、书法、绘画等诸多艺术文化印记。古典家具之美，若用中国哲学思想来概括，便是『阴阳』。其如虚实、曲直、方圆、刚柔、繁简诸般，最终达到和谐统一之态。它也像极了中国最重要的艺术——书法：字形结构、笔画之间的关系处理，最终构成一个和谐的汉字。家具亦是如此，造型比例、线条之间的位置经营，优美的气质就此诞生。

家具，构建出文人燕居行乐的世界。怀古将这些依身傍形的红木，视为文化精神之脊梁，终日投心尽力，汲取珍妙，陶冶性情，窥求传统工艺之真。

思追|古人文化

古代文化是中华民族对于人类的伟大贡献，独具特色的语言文字，浩如烟海的文化典籍，嘉惠世界的科技工艺，精彩纷呈的文学艺术，充满智慧的哲学宗教，完备深刻的道德伦理，共同构成中国文化的基本内容。在博大精深的中国古代文化中，书法、绘画、唐诗、宋词无疑是突出的代表。

古代文人以此游艺文房、治学涵养，他们以进德精艺为自豪，文化，是文人性灵最好的归宿。幽人风致，格物致知，旷世情怀。『好香用以熏德，好纸用以垂世，好笔用以生花，好墨用以焕彩』。在中国文化里，一颗文心，总能使日常的物质器皿，穿过礼乐生活，生发出璀璨的艺术之光，达天地之鉴，万物之境。


| 怀 古 |

从容闲岁月，怀古更出新

情怀|与生俱来

当下，是消费升级的盛世，也是中国文化回归的时代。人们在趋媚西方文化之后，也已经渐渐转过身来，感受到中国文化的可贵。这是必然的趋势，时空与生命都是圆的，注定有回归。

情怀不古，传统文化不是科目，也不是元素，而是长期生活的总结；不是复制过去，也不是凭空创新，而是用古人的智慧适配现代化生活。怀古从经典、自然中学会与古人对话，与今人清谈，在“慢”和“素”中能找到生活的美好本真，在“情”与“意”中构建一种生活方式。

家作|典藏时代

论起中国传统器物，皆以『器以载道』为高标，这符合先贤追求『形而上』与『形而下』完美统一的状态。纵观各朝家具之茂，明清自是此中之典范，五百余年的工序轮转，将一个近乎玄虚模糊的概念，幻化成一种真实存在，传承至今。

清风、明露，仰天地丝翎，承岁月檀雕，造就“怀古”两朝文化大成者，如“明”之潇散风逸，高韵深情；如“清”之方正端严，雍雅富丽。造型审美追慕古雅遗风，繁约得正，华实相胜；选料秉承历代古训，若孤鸿拣枝，非美不栖。雕饰装饰师出古籍古法，不吝工时，雕饰细腻，繁美精奇。传世家作，典藏巨献。
